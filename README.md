# vite-labs

## my react app
_Please note that this cannot be used with Node.js 18 or later releases_

```bash
$ cd my-react-app/
$ yarn install
```

### development

```bash
$ yarn dev
```

### build

```bash
$ yarn build
$ yarn preview
```

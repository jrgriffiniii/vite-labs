# vite-labs

## my react app
_Please note that this cannot be used with Node.js 18 or later releases_

```bash
$ cd my-react-app/
$ yarn install
```

### development

```bash
$ yarn dev
```

### build

```bash
$ yarn build
$ yarn preview
```

## my lit project
_Please note that this cannot be used with Node.js 16 or earlier releases_

```bash
$ cd my-lit-project/
$ yarn install
```

### development

```bash
$ yarn --dev install
$ yarn dev
```

#### linting the source code files

```bash
$ yarn lint
```

#### formatting the source code files (using Prettier)

```bash
$ yarn format
```

#### running the tests

```bash
$ yarn test
```

### build

```bash
$ yarn build
$ yarn preview
```
